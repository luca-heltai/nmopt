Detailed Description

template<int dim>
class KKT< dim >

Assemble and solve the all-at-once KKT system for a linear-quadratic PDE-constrained optimization problem.

The code looks for a state y, an adjoint p, and a control u minimizing

1/2 ||y-y_d||^2 + alpha/2 ||u||^2

subject to the diffusion-reaction-transport equation

-div(mu grad y) + beta . grad y + sigma y = u + f

with Dirichlet boundary conditions on state and adjoint.

The discrete unknown is ordered in three blocks:

  • block 0: state
  • block 1: adjoint
  • block 2: control

and the class assembles the full block KKT matrix so that one can access both the monolithic system and its individual blocks.
