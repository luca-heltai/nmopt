Overview¶

This lecture introduces optimal control as an optimization problem constrained by equations. We first build the finite-dimensional analogy, then move to PDE-constrained models.

1. General Optimal Control Problem¶

Choose a control variable uuu and a state variable yyy such that

(y⋆,u⋆)∈argmin⁡(y,u)J(y,u)(y^\star,u^\star)\in\operatorname*{argmin}_{(y,u)} J(y,u)(y⋆,u⋆)∈(y,u)argmin​J(y,u)
(1)

subject to

E(y,u)=0,u∈Uad.\mathcal{E}(y,u)=0,\qquad u\in\mathcal{U}_{\mathrm{ad}}.E(y,u)=0,u∈Uad​.
(2)

Key ingredients:

  • state equation E(y,u)=0\mathcal{E}(y,u)=0E(y,u)=0 (ODE/PDE/algebraic)

  • admissible controls u∈Uadu\in\mathcal{U}_{\mathrm{ad}}u∈Uad​

  • cost functional J(y,u)J(y,u)J(y,u)

  • optional extra constraints (box constraints, state constraints, etc.)

2. Forward Problem vs Control Problem¶

Forward problem:

  • data are fixed

  • solve once for yyy

Optimal control:

  • uuu is unknown

  • solve the state equation repeatedly inside optimization

Key point

In optimal control, the PDE is a constraint, not the objective.

3. Finite-Dimensional Setting (Simultaneous vs Reduced)¶

Consider

min⁡y∈Rn, u∈RmJ(y,u)s.t.Ay=Bu,\min_{y\in\mathbb{R}^n,\,u\in\mathbb{R}^m} J(y,u) \quad\text{s.t.}\quad Ay=Bu,y∈Rn,u∈Rmmin​J(y,u)s.t.Ay=Bu,
(3)

with A∈Rn×nA\in\mathbb{R}^{n\times n}A∈Rn×n invertible.

3.1 Simultaneous formulation¶

Optimize in (y,u)(y,u)(y,u) and enforce Ay=BuAy=BuAy=Bu explicitly.

3.2 Reduced formulation¶

Since AAA is invertible,

y=A−1Bu=:S(u),y=A^{-1}Bu=:S(u),y=A−1Bu=:S(u),
(4)

where SSS is the control-to-state operator. Then define

f(u):=J(S(u),u),f(u):=J(S(u),u),f(u):=J(S(u),u),
(5)

and solve

min⁡u∈Uadf(u).\min_{u\in\mathcal{U}_{\mathrm{ad}}} f(u).u∈Uad​min​f(u).
(6)

This reduces a finite-dimensional control problem to a standard finite-dimensional optimization problem.

4. Existence in Finite Dimensions¶

A standard existence result for the reduced problem:

If

  • fff is lower semicontinuous and bounded from below,

  • one level set Lt:={u∈Uad:f(u)≤t}L_t:=\{u\in\mathcal{U}_{\mathrm{ad}}:f(u)\le t\}Lt​:={u∈Uad​:f(u)≤t} is nonempty, closed, and bounded,

then a minimizer exists.

Reason: in finite dimensions, closed and bounded sets are compact (Weierstrass theorem).

Important warning

In infinite-dimensional spaces, closed and bounded sets are generally not compact. This is one of the main analytical difficulties for PDE-constrained optimization.

5. Unconstrained First/Second-Order Conditions¶

For convex differentiable fff on a convex set KKK:

∇f(uˉ)⋅(u−uˉ)≥0∀u∈K\nabla f(\bar u)\cdot (u-\bar u)\ge 0\quad\forall u\in K∇f(uˉ)⋅(u−uˉ)≥0∀u∈K
(7)

is the first-order optimality condition.

Special case (interior point):

∇f(uˉ)=0.\nabla f(\bar u)=0.∇f(uˉ)=0.
(8)

If f∈C2f\in C^2f∈C2 and uˉ\bar uuˉ is a local minimizer:

  • ∇f(uˉ)=0\nabla f(\bar u)=0∇f(uˉ)=0

  • D2f(uˉ)D^2 f(\bar u)D2f(uˉ) is positive semidefinite

If moreover D2f(uˉ)D^2 f(\bar u)D2f(uˉ) is positive definite, then uˉ\bar uuˉ is a strict local minimizer.

7. From Finite to Infinite Dimensions¶

For PDE-constrained control, state/control live in function spaces (typically Hilbert spaces):

  • state space YYY (e.g. Sobolev spaces)

  • control space UUU

  • PDE operator E(y,u)=0\mathcal{E}(y,u)=0E(y,u)=0

Typical difficulties:

  • non-compactness in infinite dimension

  • weak vs strong convergence issues

  • differentiability in Banach/Hilbert spaces

  • adjoint equations for gradient computation
