Introduction and Motivation for PDE-Constrained Optimal Control

University of Pisa

Overview¶

This lecture introduces optimal control as an optimization problem constrained by equations. We first build the finite-dimensional analogy, then move to PDE-constrained models.

Logical path of the lecture:

  1. start from a constrained optimization problem in (y,u)(y,u)(y,u);

  2. eliminate the state through the model equation when possible;

  3. obtain a reduced optimization problem in the control variable only;

  4. derive first-order conditions in finite dimension;

  5. transfer the same structure to PDE settings.

General Optimal Control Problem¶

Choose a control variable uuu and a state variable yyy such that

(y⋆,u⋆)∈argmin⁡(y,u)J(y,u)(y^\star,u^\star)\in\operatorname*{argmin}_{(y,u)} J(y,u)(y⋆,u⋆)∈(y,u)argmin​J(y,u)
(1)

subject to

E(y,u)=0,u∈Uad.\mathcal{E}(y,u)=0,\qquad u\in\mathcal{U}_{\mathrm{ad}}.E(y,u)=0,u∈Uad​.
(2)

Key ingredients:

  • state equation E(y,u)=0\mathcal{E}(y,u)=0E(y,u)=0 (ODE/PDE/algebraic)

  • admissible controls u∈Uadu\in\mathcal{U}_{\mathrm{ad}}u∈Uad​

  • cost functional J(y,u)J(y,u)J(y,u)

  • optional extra constraints (box constraints, state constraints, etc.)

Forward Problem vs Control Problem¶

Forward problem:

  • data are fixed

  • solve once for yyy

Optimal control:

  • uuu is unknown

  • solve the state equation repeatedly inside optimization

Key point

In optimal control, the PDE is a constraint, not the objective.

Finite-Dimensional Setting (Simultaneous vs Reduced)¶

Consider

min⁡y∈Rn, u∈RmJ(y,u)s.t.Ay=Bu,\min_{y\in\mathbb{R}^n,\,u\in\mathbb{R}^m} J(y,u) \quad\text{s.t.}\quad Ay=Bu,y∈Rn,u∈Rmmin​J(y,u)s.t.Ay=Bu,
(3)

with A∈Rn×nA\in\mathbb{R}^{n\times n}A∈Rn×n invertible.

Simultaneous formulation¶

Optimize in (y,u)(y,u)(y,u) and enforce Ay=BuAy=BuAy=Bu explicitly.

Interpretation:

  • optimization variable: the pair (y,u)(y,u)(y,u);

  • coupling: yyy and uuu are linked by the model equation;

  • computational consequence: every candidate pair must satisfy the constraint.

Reduced formulation¶

Since AAA is invertible,

y=A−1Bu=:S(u),y=A^{-1}Bu=:S(u),y=A−1Bu=:S(u),
(4)

where SSS is the control-to-state operator. Then define

f(u):=J(S(u),u),f(u):=J(S(u),u),f(u):=J(S(u),u),
(5)

and solve

min⁡u∈Uadf(u).\min_{u\in\mathcal{U}_{\mathrm{ad}}} f(u).u∈Uad​min​f(u).
(6)

This reduces a finite-dimensional control problem to a standard finite-dimensional optimization problem.

Step-by-step logic:

  1. the constraint Ay=BuAy=BuAy=Bu defines yyy uniquely as a function of uuu;

  2. therefore the only independent decision variable is uuu;

  3. the objective becomes a composite map u↦J(S(u),u)u\mapsto J(S(u),u)u↦J(S(u),u);

  4. all constraint information is encoded in SSS.

Existence in Finite Dimensions¶

A standard existence result for the reduced problem:

If

  • fff is lower semicontinuous and bounded from below,

  • one level set Lt:={u∈Uad:f(u)≤t}L_t:=\{u\in\mathcal{U}_{\mathrm{ad}}:f(u)\le t\}Lt​:={u∈Uad​:f(u)≤t} is nonempty, closed, and bounded,

then a minimizer exists.

Reason: in finite dimensions, closed and bounded sets are compact (Weierstrass theorem).

Why this matters for control:

  • before deriving optimality conditions, we need existence of at least one minimizer;

  • existence is easy in finite dimension under compactness of level sets;

  • this argument will fail in infinite dimensions unless additional structure is used.

Important warning

In infinite-dimensional spaces, closed and bounded sets are generally not compact. This is one of the main analytical difficulties for PDE-constrained optimization.

Unconstrained First/Second-Order Conditions¶

For convex differentiable fff on a convex set KKK:

∇f(uˉ)⋅(u−uˉ)≥0∀u∈K\nabla f(\bar u)\cdot (u-\bar u)\ge 0\quad\forall u\in K∇f(uˉ)⋅(u−uˉ)≥0∀u∈K
(7)

is the first-order optimality condition.

Special case (interior point):

∇f(uˉ)=0.\nabla f(\bar u)=0.∇f(uˉ)=0.
(8)

If f∈C2f\in C^2f∈C2 and uˉ\bar uuˉ is a local minimizer:

  • ∇f(uˉ)=0\nabla f(\bar u)=0∇f(uˉ)=0

  • D2f(uˉ)D^2 f(\bar u)D2f(uˉ) is positive semidefinite

If moreover D2f(uˉ)D^2 f(\bar u)D2f(uˉ) is positive definite, then uˉ\bar uuˉ is a strict local minimizer.

Constrained minimization (2D Example)¶

Let

Uad=R2,f(u)=12uTAu,φ(u)=Bu−g=0,\mathcal{U}_{\mathrm{ad}}=\mathbb{R}^2,\qquad f(u)=\frac12 u^T A u,\qquad \varphi(u)=Bu-g=0,Uad​=R2,f(u)=21​uTAu,φ(u)=Bu−g=0,

with

A∈R2×2 SPD,B∈R1×2,g∈R.A\in\mathbb{R}^{2\times 2}\text{ SPD},\quad B\in\mathbb{R}^{1\times 2},\quad g\in\mathbb{R}.A∈R2×2 SPD,B∈R1×2,g∈R.

Logical interpretation:

  1. objective level sets are ellipses (because AAA is SPD);

  2. feasible points lie on an affine line Bu−g=0Bu-g=0Bu−g=0;

  3. the minimizer is where the first objective level set touches that feasible line.

At an optimal feasible point, ∇f\nabla f∇f is orthogonal to the feasible tangent direction, so it must be parallel to ∇φ\nabla\varphi∇φ:

∇f(uˉ)=(∇φ(uˉ))Tλ.\nabla f(\bar u)=(\nabla\varphi(\bar u))^T\lambda.∇f(uˉ)=(∇φ(uˉ))Tλ.

Equivalent first-order form:

∇f(uˉ)−(∇φ(uˉ))Tλ=0,φ(uˉ)=0.\nabla f(\bar u)-(\nabla\varphi(\bar u))^T\lambda=0, \qquad \varphi(\bar u)=0.∇f(uˉ)−(∇φ(uˉ))Tλ=0,φ(uˉ)=0.

Meaning:

  • φ(uˉ)=0\varphi(\bar u)=0φ(uˉ)=0: feasibility at the solution;

  • gradient balance: objective gradient is compensated by constraint normal direction;

  • multiplier λ\lambdaλ: strength/sign of that compensation.

Lagrangian Formalism¶

Define the Lagrangian

∇uL=Au−BTλ=0,∇λL=−Bu+g=0,\nabla_u\mathcal{L}=Au-B^T\lambda=0, \qquad \nabla_\lambda\mathcal{L}=-Bu+g=0,∇u​L=Au−BTλ=0,∇λ​L=−Bu+g=0,

(A−BT −B0)(u λ)=(0 −g).\begin{pmatrix}A & -B^T \\\ -B & 0\end{pmatrix}\begin{pmatrix}u \\\ \lambda\end{pmatrix}=\begin{pmatrix}0 \\\ -g\end{pmatrix}.(A −B​−BT0​)(u λ​)=(0 −g​).

This is the prototype for all later optimality systems:

  • state/constraint equation;

  • adjoint or multiplier equation;

From Finite to Infinite Dimensions¶

For PDE-constrained control, state/control live in function spaces (typically Hilbert spaces):

  • state space YYY (e.g. Sobolev spaces)

  • control space UUU

  • PDE operator E(y,u)=0\mathcal{E}(y,u)=0E(y,u)=0

Typical difficulties:

  • non-compactness in infinite dimension

  • weak vs strong convergence issues

  • differentiability in Banach/Hilbert spaces

  • adjoint equations for gradient computation

Conceptual continuity with the finite-dimensional case:

  1. same optimization structure;

  2. same reduced-vs-simultaneous viewpoints;

  3. same KKT logic;

  4. only the functional-analytic setting changes.

Typical Variations of OCP Formulations¶

Many optimal control models keep the same abstract structure but vary in where the control acts, what is observed, and how the objective is measured.

Common variations include:

  • Localized distributed control: uuu acts only on a subdomain ω⊂Ω\omega\subset\Omegaω⊂Ω (for example through χωu\chi_\omega uχω​u in the PDE).

  • Boundary control: uuu appears in Dirichlet/Neumann/Robin boundary conditions on part of ∂Ω\partial\Omega∂Ω.

  • Initial-condition control: uuu is an unknown initial datum in time-dependent models.

  • Parameter control: uuu is a coefficient (diffusivity, reaction rate, material parameter), not a source term.

  • Different tracking norms: replace L2L^2L2 tracking by H1H^1H1, weighted norms, or mixed space-time norms.

  • Different control penalties: use L2L^2L2, H1H^1H1, or sparsity-promoting terms (e.g. L1L^1L1-type penalties).

  • Pointwise state constraints: enforce bounds on the state (or output) in the domain or on the boundary.

  • Multi-objective costs: combine tracking, regularization, and engineering criteria (energy, drag, flux, etc.).

  • Inverse problems / data assimilation: identify unknown inputs/parameters from measurements with regularization.

  • Uncertainty-aware OCPs: optimize expected cost, risk measures, or robust worst-case criteria.

These variants motivate why we need a flexible theoretical framework and multiple numerical methods in the rest of the course.
