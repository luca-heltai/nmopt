What Is Optimal Control?¶

  • Choose a control uuu to minimize a cost J(y,u)J(y,u)J(y,u).

  • The state yyy is constrained by a model (often a PDE).

  • Optimization + differential constraints = PDE-constrained optimization.

Abstract Formulation¶

Minimize

min⁡y,u  J(y,u)\min_{y,u} \; J(y,u)y,umin​J(y,u)
(1)

subject to the state equation

E(y,u)=0.\mathcal{E}(y,u) = 0.E(y,u)=0.
(2)

Typical pieces:

  • state equation (PDE)

  • admissible controls u∈Uadu \in \mathcal{U}_{\mathrm{ad}}u∈Uad​

  • constraints (box constraints, etc.)

Forward vs Optimal Control¶

  • Forward (direct) problem:

    • input data given

    • solve PDE once to compute yyy

  • Optimal control:

    • control uuu unknown

    • PDE is solved repeatedly while optimizing JJJ

Key point: the PDE is a constraint, not the objective.

Reduced Formulation¶

If the state equation defines a control-to-state map SSS:

y=S(u),y = S(u),y=S(u),
(3)

then define the reduced cost

f(u):=J(S(u),u),f(u) := J(S(u),u),f(u):=J(S(u),u),
(4)

and solve

min⁡uf(u).\min_{u} f(u).umin​f(u).
(5)

Why it matters:

  • separates “PDE solve” from “optimization step”

  • makes gradients/adjoins systematic

When PDEs Show Up¶

States: temperature, displacement, velocity, concentration, ...

Controls:

  • distributed sources

  • boundary inputs

  • initial conditions

  • parameters / coefficients

Typical Examples¶

  • Elliptic: stationary heating / diffusion

  • Parabolic: time-dependent heating

  • Nonlinear: Navier-Stokes flow control

  • Inverse problems: parameter estimation / data assimilation

Constraints¶

Common: box constraints on the control

umin⁡≤u≤umax⁡.u_{\min} \le u \le u_{\max}.umin​≤u≤umax​.
(13)

Leads to:

  • variational inequalities

  • KKT conditions

  • active-set / projection-type algorithms
